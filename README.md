## Development:
Requirements: php with composer, mysql, yarn
### Frontend:
* `cd frontend`
* `yarn install` to get packages
* `yarn run start` to start the webpack dev server on :3000
* `yarn run test` to start the test suite watcher
### Backend
* `cd backend`
* `composer install`
